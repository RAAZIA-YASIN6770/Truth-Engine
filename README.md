# Truth Engine

**Browser-based Logical Reasoning System**

## 1. Overview
Truth Engine is a single-player web application designed to evaluate logical consistency and contradiction detection. It operates on a deterministic rule-based engine, strictly adhering to IEEE 830 SRS standards.

## 2. Project Scope
*   **Core Function:** Logical contradiction scenarios and deterministic rule evaluation.
*   **Architecture:** Client-Server (Python/Flask Backend + Browser Frontend).
*   **Methodology:** Iterative Incremental Model.

## 3. Technical Stack
*   **Backend:** Python 3.10+ (Flask)
*   **Frontend:** HTML, CSS, JavaScript
*   **Version Control:** Git

## 4. Setup & Installation
1.  Initialize a virtual environment (optional but recommended).
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the application:
    ```bash
    python main.py
    ```
